# jwt-auth

## Installation
Use [yarn](https://yarnpkg.com/) as your package manager
```
cd frontend
yarn
```
```
cd backend
yarn
```
Change fields in .env file to your own`s like in example below:
```
PORT=5000  
MONGO_URI="your mongo uri"
JWT_SECRET="your jwt secret"
JWT_EXPIRATION="your jwt expiration"
```

## Run
Frontend:
```
cd frontend
yarn start
```
Backend:
```
cd backend
yarn start
```
